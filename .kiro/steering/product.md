# Product Overview

Firebase Storage上のアートワーク画像を、雰囲気・色・モチーフなどの自然言語キーワードでセマンティック検索するシステム。ローカル環境（Appleシリコン）で軽量VLMを活用し、オフラインの特徴抽出とオンラインのベクトル検索を分離する。

## Core Capabilities

1. **オフラインインジェスション**: 画像からキャプション・タグ・埋め込みベクトルを自動抽出しQdrantに保存
2. **セマンティック検索**: 自然言語クエリを構造化分解し、payloadフィルタ＋ベクトル類似度検索を組み合わせる
3. **ヒット理由表示**: なぜその作品がヒットしたかをユーザーに説明する
4. **Taxonomy管理**: VLM出力の語彙揺れを正規化し、一貫性のあるメタデータを維持する

## Target Use Cases

- アート作品コレクションから「やさしい感じで、緑と金が入っていて、空っぽい作品」のような直感的な日本語クエリで検索
- 新規作品追加時のメタデータ自動生成と検索インデックス登録
- 既存画像ライブラリの一括インジェスション（1日1回以上のバッチ処理）

## Value Proposition

- SigLIP2による多言語対応のセマンティック検索で、タグ完全一致では見つからない「雰囲気」検索を実現
- ベクトル検索（雰囲気）とpayloadフィルタ（色・モチーフ）の役割分担により、精度と柔軟性を両立
- ローカルモデル運用によりAPI課金なし、プライバシー確保
